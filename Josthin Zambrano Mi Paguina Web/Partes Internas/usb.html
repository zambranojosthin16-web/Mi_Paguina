🔌 Puerto USB

El Puerto USB (Universal Serial Bus) es una interfaz estándar que permite la conexión de diferentes dispositivos periféricos a la computadora, como teclados, ratones, memorias flash, impresoras, cámaras, entre otros.

Caracteristicas de la USB

  • Conexión: Permite conectar dispositivos externos.
  • Transferencia: Transfiere datos rápidamente.
  • Compatibilidad: Funciona con muchos dispositivos.
